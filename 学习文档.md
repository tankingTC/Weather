# Weather 天气助手 — 完整项目学习文档

***

## 一、项目总览

### 1.1 项目简介

这是一个 Android **天气助手** 应用，使用 **Java** 编写，基于 **和风天气 API** 获取实时天气、7日预报和空气质量数据，集成 **百度地图 SDK** 实现地图选点定位，以及 **阿里云百炼（通义千问）AI 大模型** 提供智能穿衣建议和 AI 天气问答功能。

### 1.2 技术栈一览

| 类别     | 技术/库                            | 用途                  |
| ------ | ------------------------------- | ------------------- |
| 语言     | Java                            | 应用主体逻辑              |
| 构建工具   | Gradle (Kotlin DSL) + AGP 9.0.1 | 项目构建与依赖管理           |
| 网络请求   | Retrofit 2.9 + OkHttp 4.12      | 和风天气/百度地图/百炼 API 通信 |
| JSON解析 | Gson 2.10                       | JSON 序列化/反序列化       |
| 本地数据库  | Room 2.6.1                      | 持久化存储已收藏城市          |
| 本地缓存   | SharedPreferences               | 存储选中城市、天气缓存、上次定位    |
| 图片加载   | Coil 2.7 / Glide 4.16           | 天气图标/资源加载           |
| 地图     | 百度地图 SDK 7.6.4                  | 地图选点、逆地理编码          |
| AI 大模型 | 阿里云百炼 (DashScope)               | 穿衣建议生成 + AI 天气问答    |
| UI     | ViewBinding + Material Design 3 | 现代化 UI 组件           |
| 异步     | ExecutorService + Handler       | 后台任务调度              |
| 架构模式   | Repository 模式                   | 数据层封装               |

### 2.2 Repository 模式

项目采用 **Repository 模式** 将数据获取逻辑与 UI 层完全分离：

- [WeatherRepository](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/service/WeatherRepository.java#L29)：负责天气数据、预报数据、空气质量数据的获取，以及城市搜索和坐标反查。内置缓存策略（先查缓存，网络可用时请求最新数据）。
- [AiAssistantRepository](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/service/AiAssistantRepository.java#L23)：负责与阿里云百炼 AI 模型通信，支持两种模式：Agent App 模式和通用 Chat Completion 模式（降级策略）。
- [ClothingSuggestionService](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/service/ClothingSuggestionService.java#L10)：纯逻辑服务，根据天气数据本地计算穿衣建议。

## 四、数据模型（Model）详解

### 4.1 WeatherResponse — 实时天气

**文件**: [WeatherResponse.java](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/model/WeatherResponse.java)

对应和风天气 `v7/weather/now` 接口。内部类 `Now` 包含：

- `temp`：当前温度
- `feelsLike`：体感温度
- `icon`：天气图标代码（如 100=晴，305=大雨）
- `text`：天气文字描述
- `windDir`/`windScale`/`windSpeed`：风向/风力/风速
- `humidity`：湿度百分比
- `precip`：降水量(mm)
- `pressure`：气压(hPa)
- `vis`：能见度(km)
- `cloud`：云量百分比
- `dew`：露点温度

### 4.2 ForecastResponse — 7日预报

**文件**: [ForecastResponse.java](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/model/ForecastResponse.java)

对应 `v7/weather/7d` 接口。`Daily` 内部类包含：

- `fxDate`：预报日期
- `tempMax`/`tempMin`：最高/最低温度
- `iconDay`：白天天气图标代码
- `textDay`：白天天气描述

### 4.3 AirQualityResponse — 空气质量

**文件**: [AirQualityResponse.java](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/model/AirQualityResponse.java)

对应和风天气空气质量接口。`Index` 含 AQI 数值、类别、主要污染物、健康建议等。

### 4.4 City — 城市实体

**文件**: [City.java](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/model/City.java)

使用 **Room @Entity** 注解，对应数据库 `cities` 表：

- `@PrimaryKey id`：城市 ID
- `name`：城市名
- `lat`/`lon`：经纬度
- `addTime`：添加时间（用于排序）
- `currentTemp`/`weatherText`：`@Ignore` 临时字段（不入库，仅在列表展示时临时使用）

### 4.5 CityResponse — 城市搜索结果

**文件**: [CityResponse.java](file:///d:/Android/project/Weather/app/src/main/java/com/example/weather/model/CityResponse.java)
